| /pause   | Pause the audio stream.                              |
| /resume  | Resume the audio stream.                             |
| /mute    | Mute the userbot.                                    |
| /unmute  | Unmute the userbot.                                  |

| /restart | Restart the bot (Only for bot Owner).                |
